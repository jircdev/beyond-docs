# Comando: beyond add package

Crear un nuevo paquete con BeyondJS.
Este comando crea una estructura de directorios y archivos para un paquete en BeyondJS, según el tipo de paquete y una serie de parámetros que defina el usuario.

## Parámetros

- **Specifier**: Especificador del paquete.
- **Title**: Título del paquete.
- **About**: Descripción del paquete.
- **Package type**: es el nombre de la plantilla que se desea utilizar para el paquete.

### Tipos de paquetes en BeyondJS:

web: proporciona una estructura básica para una aplicación web.
backend: proporciona una estructura básica para una aplicación de backend en Node.js.
node: proporciona una estructura básica para un proyecto de Node.js.
express: proporciona una estructura básica para una aplicación web utilizando el framework Express en Node.js.
react: proporciona una estructura básica para un paquete con react.
svelte: proporciona una estructura básica para un paquete con svelte.
vue: proporciona una estructura básica para un paquete con vue.
web-backend: proporciona una estructura básica para una aplicación web con un backend en Node.js.
empty: proporciona una estructura mínima requerida para un paquete en BeyondJS.
