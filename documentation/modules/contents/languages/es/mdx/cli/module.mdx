# Comando: beyond add module

Crear un módulo específico dentro de un paquete hecho con BeyondJS.
Genera una estructura de directorios y archivos para un módulo dentro de un paquete especificado.

## Parámetros

- **package**: Nombre del paquete en el cual se creará el módulo.
- **subpath**: Subpath o name del módulo a crear.
- **Module type**: Representa el tipo de módulo, puede ser: "page", "widget", "layout", "code", "ts", "start".
- **Web component name**: Define el nombre del web component sí es un módulo tipo "page", "widget" o "layout".
- **url**: Define la URL para un módulo tipo page.
- **about**: Descripción del módulo.
- **styles**: Indica si el módulo tiene hojas de estilo.
- **Multilanguage**: Indica si el módulo tiene multilenguaje.
