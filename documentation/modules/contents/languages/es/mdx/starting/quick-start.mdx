#### Prerequisitos

-   Node ^14.17.4.
-   Estar familiarizado con manejadores de dependencias como npm o yarn.

<Subtitle>beyond-first-project</Subtitle>

Para empezar, posiblemente la manera más sencilla de arrancar un proyecto en BeyondJS sea por medio del `beyond-first-project`.

<Code>{`npx beyond-first-project -name @scope/name [-type node] [-container folderName]`}</Code>

Opciones del comando:

-   name: Permite definir el **scope y nombre** del proyecto. Los proyectos, siguen el estándar de **NPM**, puedes leer más
    sobre ello <Link to="/docs/intro#projects">aquí.</Link>
-   type: Permite definir el tipo de proyecto a crear, es opcional, sino se pasa se crea un proyecto de tipo web por defecto.
    Los valores aceptados son:

        -   web
        -   node
        -   backend
        -   library
        -   web-backend

Puedes leer más información sobre los tipos de proyecto <Link to="/docs/projects/types">aquí.</Link>

-   container: Permite especificar si se desea agregar una carpeta contenedora de proyectos. Recuerde que BeyondJS soporta la posibilidad de manejar múltiples proyectos en simultáneo.

Para ejemplificar, se puede imaginar que se necesita crear un proyecto login de tipo **node**, lo podríamos crear con el siguiente comando:

<Code>{`npx create-first-project -name @scope/name -type node`}</Code>

Al finalizar la ejecución del comando, se debió crear la estructura del proyecto y ya estaría listo para ser ejecutado.

Para levantar el servidor de desarrollo, se ejecuta el siguiente comando:

<Code>{`npm start`}</Code>

¡Listo! Ya se puede comenzar a programar y probar el proyecto. Se puede acceder al Workspace desde <ELink to="https://workspace.beyondjs.com/">https://workspace.beyondjs.com/</ELink> y el proyecto creado se encuentra disponible en <ELink to="http://localhost:6500">http://localhost:6500</ELink>

El Workspace es un sistema que BeyondJS provee para brindar al desarrollador una mejor experiencia de usuario. Puedes leer más sobre el <Link to="/docs/workspace/intro">aqui.</Link>

-   <Link to="/docs/projects/structure">Estructura de un proyecto</Link>
-   <Link to="/docs/modules/intro">Modules</Link>

<Subtitle>Instalación global</Subtitle>

Para instalar BeyondJS globalmente, se debe correr el siguiente comando:

<Code>{`npm i -g beyond `}</Code>

<Code>{`beyond run [--workspace=port]`}</Code>

> Puedes obtener más información sobre el CLI de <BeyondName/> <Link href="/cli">en esta página</Link>.

Si todo sale bien, aparecerá un mensaje diciendo que BeyondJS se está ejecutando y que puedes acceder al Workspace
en <ELink to="https://workspace.beyondjs.com/">https://workspace.beyondjs.com/</ELink>, desde donde se podrá crear un
proyecto por medio de la interfaz gráfica del Workspace.

Ya en el Workspace, puedes crear un proyecto desde la interfaz gráfica, puedes ver cómo
hacerlo <Link to="/docs/projects/create">aquí</Link>.
