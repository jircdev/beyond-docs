<DocHeader pretitle="Introducción" title="Comienzo rápido" />

<Info>

Para entender esta documentación debes tener en cuenta los siguientes puntos.

    -  Tener instalado Node ^14.17.4.
    -  Estar familiarizado con manejadores de dependencias como npm o yarn.

</Info>

<BeyondJS /> puede ser instalado por medio de dos comandos:

-   Instalación global con npm `npm i -g beyond`.
-   Instalación y creación de un proyecto con npx `npx @beyond-js/create-package`.

<Title tag="h3" text="npx @beyond/create-package" />

Este es el comando ideal sino tienes instalado <BeyondJS/>, ya que realiza la instalación global
y procede a crear un paquete de forma automática en el directorio en el que se ejecuta.

Es un atajo para ejecutar la instalación de <BeyondJS/> y el comando `beyond add package` que permite crear un paquete.

Al ejecutarlo, te consultará por medio de la consola interactiva el nombre del paquete a crear y el tipo de paquete, una vez seleccionados,

<BeyondJS /> terminará el proceso de instalación y ya estás listo para poder ejecutar tu proyecto y empezar a programar.

A continuación, solo necesitas ejecutar el siguiente comando en el directorio raiz del proyecto.

` beyond run --workspace 4000`

Este comando inicializa BeyondJS y te permite navegar a tu proyecto en <ELink href="http://localhost:6530"> http://localhost:6530</ELink>.

> El puerto `6530` es definido por defecto en la distribución de proyecto y puede modificarse de forma simple en el `package.json`. Puedes leer en detalle
> sobre ello en <Link href="/distributions"> la sección de distribuciones</Link>

<Title tag="h3" text="npm -g beyond" />

Este comando realiza una instalación global que permite ejecutar el <Link href="/dev-server">DevServer</Link> de

<BeyondJS /> en el directorio que se desee y habilita la lista de comandos disponibles en el CLI.

> Para ver la lista de comandos de <BeyondJS/> puedes ir a <Link href="/cli"> la sección del CLI</Link>

Con <BeyondJS/> instalado, puedes proceder a crear un proyecto en la carpeta que desees ejecutando el siguiente comando:

```
beyond add project
```

Este comando te preguntará por el nombre y tipo de proyecto que deseas crear, entre los que puedes seleccionar:

-   Web
-   Node
-   Backend
-   web-Backend
-   express.
