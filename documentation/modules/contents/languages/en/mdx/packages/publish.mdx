To publish a package in NPM you must currently add an npm distribution in the `package.json`.
The distribution must contain a `platforms` property with an object indicating the platforms you want to compile and the distribution associated with this platform.
to this platform.

### Example

<Code>
{`
{
  "name": "npm",
  "npm": {
    "platforms": {
      "web": "web"
    }
  }
}
`}

</Code>
