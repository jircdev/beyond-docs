<Title>BeyondJS Dependencies</Title>

BeyondJS offers a list of required dependencies depending on the type of project you want to create.

<Subtitle>**@beyond/kernel**</Subtitle>

It is the main library of BeyondJS, it contains modules to cover the basic functionalities of the Framework. It contains the following packages:

-   **bundle:** used by any <Link href="/docs/bundles">bundle</Link>, common functions for them.
-   **core:** manages the universal package import handling.
-   **routing:** Contains all functionalities associated with the project's history and navigation.
-   **styles:** Serves to work with style sheets in a reactive way.
-   **texts:** required for the management of multilingual functionalities.

<Subtitle>**@beyond/backend**</Subtitle>

Library that manages the client and server side of real-time connection and bridges. Required in **node** and **backend** type projects.

<Subtitle>**@beyond/ssr**</Subtitle>

It is responsible for providing the APIs to access the rendering information of the widgets and make the server rendering available.

<Subtitle>Libraries for widgets</Subtitle>

To work with widgets it is necessary to have the @beyond/widgets dependency installed and additionally, depending on the framework you want to use, BeyondJS offers the following libraries:

-   @beyond/react-widget
-   @beyond/vue-widget
-   @beyond/svelte-widget

All the mentioned libraries can be installed through the command npm i library where "library" would be the name of the library to be installed.
