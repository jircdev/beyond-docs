<Title>Module configuration</Title>.

The module configuration file is module.json.

<Code>
{`
{
  "name": "home",
  "platforms": [
    "backend",
    "web"
  ],
  "widget": {
    "hmr": true,
    "element": {
      "name": "home-page"
    },
    "name": "home",
    "route": "/${content}",
    "is": "page",
    "ts": {
      "path": "/widget/ts",
      "files": [
        "*"
      ]
    }
  }
}

`}

</Code>

The configuration properties are:

-   **name** [string] : Defines the name of the module, necessary for the elements to be imported.
-   platforms\*\* [array] : Defines the platforms supported by the module. The supported values are the identifiers of the existing <Link href="/concepts/platforms">platforms</Link>.
