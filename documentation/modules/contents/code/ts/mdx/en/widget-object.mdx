<Title>Objeto BeyondWidget</Title>

The BeyondWidget object is the parent object of widgets and inherits directly from the <ELink to="https://developer.mozilla.org/es/docs/Web/API/HTMLElement">HTMLElement</ELink> object and provides an API that allows you to interact with the BeyondJS widget tree.

<SubTitle>Properties</SubTitle>

-   **wnode <Link to="/docs/api/NodeWidget">NodeWidget</Link> optional:** Widget HTML node.
-   **wchildren <Link to="/docs/api/BeyondWidget">(BeyondWidget)[]</Link> optional:** List of contained widgets
-   **parent <Link to="/docs/api/BeyondWidget">BeyondWidget</Link> optional:** Widget container, returns undefined if a widget does not exist as a container.
