<SubTitle>EOC</SubTitle>

("External Object Configuration"). Configuration object that can be included directly as a value or referenced as an external file. Objects whose properties are EOCs can expect two values: an object or a string. If the property has a string as its value, BeyondJS takes the value as a relative path to the location of the external configuration file.
